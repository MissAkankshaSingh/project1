# project1
This currency converter project showcases the power
of javaScript in creating dynamic and interactive web
applications. By utilizing real-time currency exchange
rates, users can effotlessly convert between various
currenices.

//Features:

User-Friendly Interface: The project offers an intutive
interface that makes it easy for users to input the 
amount and select the source and target currencies
for conversion.

Real-Time Exchange Rates: Leveraging APIs or libraries,
the application fetches real-time exchanges rates from
reliable sources, ensuring accurate and up-to-date
conversions.
